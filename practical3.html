The ways to open developers console in Mozilla Firefox and Safari

Firefox

There are two ways to open developer tools in Mozilla Firefox:

  1. Right-click on the page, then choose Inspect Element from the menu and after that select the Console tab. (or ctrl+click for Apple touchpads/magic mouse devices)
  2. Use the shortcut. You should press Cmd+Option+K (for Mac) or Ctrl+Shift+J (Windows) on your keyboard.

Safari

To open the developer console on the Safari you need to make the Develop menu visible. To do it, you'll need to go to Safari > Preferences > Advanced and enable Show Develop menu in menu bar.

Then, you can do one of the following:

  • Right-click on the page, select Inspect Element and then choose the Console tab. (or ctrl+click for Apple touchpads/magic mouse devices)
  • Use the hotkey combination Cmd + Option + C

Use cases for choosing absolute links and choosing relative links

Absolute URLs

An absolute URL provides a complete and specific location of a webpage. It includes the entire web address, starting from the protocol to the page name. For example, “https://example.com/blog/article.html” is an absolute URL.

Relative URLs

In contrast, a relative URL provides a shorter, more flexible address that depends on the context within the website. It only includes the path to the webpage relative to the current location. For instance, if we are on the “https://example.com/blog/” page, a relative URL for the article would be “/blog/article.html.”

Use cases for choosing absolute links:

  • When linking to resources on external websites
  • When linking to resources on different subdomains or domains within the same website
  • When the website needs to maintain consistent links even if the site's URL structure changes

Use cases for choosing relative links:

  • When linking to resources within the same website
  • When building a website with a consistent URL structure, where the links need to adapt automatically to changes in the site's domain or subdomain

What way of including internal links into the HTML page is more preferable?

An absolute URL — while potentially more work upfront — is easier to maintain in the long run. Not only is it easier for the search engines to understand, but it prevents links from easily breaking, prevents duplicate content issues and preserves the crawl budget.

Image formats used in web-development

JPG or (JPEG), GIF, and PNG are the main graphic formats used online. These file formats are also used in print, multimedia, and presentations.

What is the difference between JPG, GIF and PNG formats?

JPG

JPG or JPEG (Joint Photographic Experts Group) is the main file format used for online and printed images. This file type is great for photographs as it supports a full spectrum of colours. It also displays well on all devices and can be easily saved.

One problem is that while you can reduce the file size of JPGs, the quality of the image will lessen as you compress the file. It is therefore not great for images that need to be continuously edited.

GIF

GIF (Graphics Interchange Format) is the main file format used for animation. It is the only way to create online animations without using Flash.

GIF only supports 8-bit colours meaning you can only use 256 colours.

GIF does support background transparency and can be used for line graphics, icons, and logos as well as for basic animations. Unlike JPG, image quality will not degrade with editing and re-saves.

GIF also has the smallest file size of all three file formats so should not affect website load time at all if uploading logos or other designs to a site.

PNG

PNG (Portable Network Graphics) was designed as an improved alternative to GIF. It is great for internet graphics and supports transparency in a way that GIF files do not.

PNG files support 24-bit colours, which includes 16.7 million colours, and features background transparency. Image quality will also not decrease no matter how many times the image is compressed or edited and re-saved.

Unlike GIF, PNG does not support animation.

What is WEBP image type?

WebP is an open-source image file format developed by Google that supports both lossy and lossless image compression. A WebP file is an image file in the WebP format. First introduced in 2010, the WebP format has been gaining traction online in recent years and was created to deliver high-quality, lightweight image files over the web.

What is SVG?

SVG is a scalable vector graphics markup language created by the World Wide Web Consortium and is a subset of the Extensible Markup Language XML. SVG is used to define vector-based graphics for the Web. SVG defines graphics in XML format. Each element and attribute in SVG files can be animated.
